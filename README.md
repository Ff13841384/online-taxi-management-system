# Online Taxi Management System

A Java console-based application that simulates an online taxi service.

## Features

- Request a taxi ride
- Passenger count validation
- Random driver generation
- Driver selection
- Distance calculation
- Fare calculation
- Weather-based pricing
- Discount code support
- Driver rating system

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Random
- Math Library

## Project Structure

- Passenger
- Driver
- TaxiManager

## How to Run

```bash
javac First.java
java First
```
